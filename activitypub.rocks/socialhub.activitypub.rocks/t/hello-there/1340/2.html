Hi everyone!

I thought it would be a good thing to come here and say “hi” since 2021 will be the year where I’ll be going more deeper in the ActivityPub space.

I have developed some tools around Mastodon API in the past, the most notable might be the multi-accounts focused client app Sengi.

Currently I’m developing a pet-project on ActivityPub to understand a bit more how all this is working. It’s just a stupid Twitter2ActivityPub relay. The code is awful but it’s functional, I have a live instance here and the code is linked in the page (can’t post more than 2 links as a new user).
I will refine and refactor the code during the following weeks so that I’ll have a healthy base to work from.

My aim is to develop some more ambitious and useful tools in the close future, since ActivityPub can support ideas I had since some years already. I will try to develop my vision first in the first half of 2021 and will work on it concretely just after.
But one step at the time, for now I’m the stupid padawan here to make a lot of errors and hopefully to learn from it.

See you around!
