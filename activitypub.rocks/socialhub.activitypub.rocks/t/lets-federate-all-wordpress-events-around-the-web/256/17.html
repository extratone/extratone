Emphack November 2, 2019, 2:14pm #1

Several projects are moving towards having federated events:

  • https://gancio.org
  • https://joinmobilizon.org
  • GetTogether
  • WeChange

According to HostingTribunal WordPress powers 34% of Internet websites (), which often has a calendar using some plug-in like Events Manager. What about allowing all these events to get federated?

We already have two plugins to federate WordPress articles, maybe they can be a starting point to federate also events?

  • https://wordpress.org/plugins/activitypub/
  • https://wordpress.org/plugins/pterotype/

The best IMO would be create a plugin compatible with the most used events managers.
