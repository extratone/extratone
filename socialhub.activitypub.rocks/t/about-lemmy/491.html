Lemmy is a link aggregator / reddit alternative, intended to work in the fediverse.

It’s self-hostable, has live-updating comment threads, and is tiny ( ~80kB ). Federation into the ActivityPub network is on the roadmap.

Suggest new features or report bugs at https://github.com/LemmyNet/lemmy.

See live instances:

  • https://dev.lemmy.ml

Test federation:

  • https://voyager.lemmy.ml
  • https://enterprise.lemmy.ml
  • https://ds9.lemmy.ml

Made with Rust, Actix, Inferno, Typescript.
