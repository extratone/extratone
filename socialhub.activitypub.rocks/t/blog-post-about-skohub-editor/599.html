Blog post about SkoHub Editor

Software SkoHub
acka47 March 31, 2020, 12:34pm #1

See http://blog.lobid.org/2020/03/31/skohub-editor.html

In a nutshell, SkoHub Editor enables the automatic generation of a web form based on a JSON schema, along with the possibility to look up terms in a controlled vocabulary that is published with SkoHub Vocabs. Additionally, metadata generated by the editor can be published using the ActivityPub-based SkoHub PubSub
